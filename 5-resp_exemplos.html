🧩 Desafio 1: Classe Produto (com encapsulamento)

📌 Classe: DesafioProduto.java
📌 Atributos privados: nome, preco, quantidade
📌 Getters e Setters para cada atributo
📌 Método aplicarDesconto(double percentual) com validação (0-100%)
📌 Método exibirInfo() para mostrar todos os dados
📌 main com testes de valores válidos e inválidos
📚 TRÍADE DIDÁTICA:
  • CONCEITO: Encapsulamento protege os dados do produto.
  • OBJETIVO: Garantir que nome, preço e quantidade sejam sempre válidos.
  • MÉTODO: Atributos private e setters com validação.
📤 Saída esperada:
--- TESTE: Valores válidos ---
Produto: Notebook
Preço: R$ 3500.0
Quantidade: 10

--- APLICANDO DESCONTO ---
Desconto de 10% aplicado!
Novo preço: R$ 3150.0

--- TESTE: Valores INVÁLIDOS ---
❌ Preço inválido! Deve ser maior que zero.
❌ Quantidade inválida! Deve ser maior que zero.
❌ Desconto inválido! Use entre 0 e 100.

📝 Código Sugerido:

package tema07;

public class DesafioProduto {
    // ATRIBUTOS PRIVADOS
    private String nome;
    private double preco;
    private int quantidade;

    // ========== GETTERS ==========
    public String getNome() {
        return nome;
    }

    public double getPreco() {
        return preco;
    }

    public int getQuantidade() {
        return quantidade;
    }

    // ========== SETTERS (com validação) ==========
    public void setNome(String nome) {
        if (nome != null && !nome.trim().isEmpty()) {
            this.nome = nome;
        } else {
            System.out.println("❌ Nome inválido!");
        }
    }

    public void setPreco(double preco) {
        if (preco > 0) {
            this.preco = preco;
        } else {
            System.out.println("❌ Preço inválido! Deve ser maior que zero.");
        }
    }

    public void setQuantidade(int quantidade) {
        if (quantidade >= 0) {
            this.quantidade = quantidade;
        } else {
            System.out.println("❌ Quantidade inválida! Deve ser maior que zero.");
        }
    }

    // ========== MÉTODOS ==========
    public void aplicarDesconto(double percentual) {
        if (percentual >= 0 && percentual <= 100) {
            double desconto = preco * (percentual / 100);
            preco -= desconto;
            System.out.println("Desconto de " + percentual + "% aplicado!");
            System.out.println("Novo preço: R$ " + preco);
        } else {
            System.out.println("❌ Desconto inválido! Use entre 0 e 100.");
        }
    }

    public void exibirInfo() {
        System.out.println("Produto: " + nome);
        System.out.println("Preço: R$ " + preco);
        System.out.println("Quantidade: " + quantidade);
    }

    // ========== MAIN PARA TESTE ==========
    public static void main(String[] args) {
        DesafioProduto produto = new DesafioProduto();

        System.out.println("--- TESTE: Valores válidos ---");
        produto.setNome("Notebook");
        produto.setPreco(3500.0);
        produto.setQuantidade(10);
        produto.exibirInfo();

        System.out.println("\n--- APLICANDO DESCONTO ---");
        produto.aplicarDesconto(10);

        System.out.println("\n--- TESTE: Valores INVÁLIDOS ---");
        produto.setPreco(-100);      // ❌ Bloqueado!
        produto.setQuantidade(-5);   // ❌ Bloqueado!
        produto.aplicarDesconto(150); // ❌ Bloqueado!
    }
}
🎯 O QUE FOI APRENDIDO:
  • ✅ Uso de private para proteger atributos
  • ✅ Getters para ler valores com segurança
  • ✅ Setters com validação para evitar dados inválidos
  • ✅ Método com validação (aplicarDesconto)

🧩 Desafio 2: Classe ContaBancaria

📌 Classe: DesafioConta.java
📌 Atributos privados: titular, numero, saldo
📌 Getters para todos, setters apenas para titular e número
📌 Métodos: depositar(double valor) e sacar(double valor)
📌 Método exibirSaldo() para mostrar os dados
📌 main com testes de depósitos, saques e validações
📚 TRÍADE DIDÁTICA:
  • CONCEITO: Encapsulamento protege o saldo da conta.
  • OBJETIVO: Garantir que apenas operações válidas (depósito/saque) alterem o saldo.
  • MÉTODO: Atributo saldo privado, modificado apenas por métodos depositar e sacar.
📤 Saída esperada:
--- TESTE: Operações válidas ---
Titular: Maria Santos
Número: 12345
Saldo: R$ 0.0

Depósito de R$ 1000.0 realizado!
Saldo atual: R$ 1000.0

Saque de R$ 300.0 realizado!
Saldo atual: R$ 700.0

--- TESTE: Operações INVÁLIDAS ---
❌ Valor de depósito inválido!
❌ Saldo insuficiente!
Saldo atual: R$ 700.0

📝 Código Sugerido:

package tema07;

public class DesafioConta {
    // ATRIBUTOS PRIVADOS
    private String titular;
    private int numero;
    private double saldo;

    // ========== GETTERS ==========
    public String getTitular() {
        return titular;
    }

    public int getNumero() {
        return numero;
    }

    public double getSaldo() {
        return saldo;
    }

    // ========== SETTERS (apenas para titular e número) ==========
    public void setTitular(String titular) {
        if (titular != null && !titular.trim().isEmpty()) {
            this.titular = titular;
        } else {
            System.out.println("❌ Titular inválido!");
        }
    }

    public void setNumero(int numero) {
        if (numero > 0) {
            this.numero = numero;
        } else {
            System.out.println("❌ Número da conta inválido!");
        }
    }

    // ========== MÉTODOS ==========
    public void depositar(double valor) {
        if (valor > 0) {
            saldo += valor;
            System.out.println("Depósito de R$ " + valor + " realizado!");
            System.out.println("Saldo atual: R$ " + saldo);
        } else {
            System.out.println("❌ Valor de depósito inválido!");
        }
    }

    public void sacar(double valor) {
        if (valor > 0 && valor <= saldo) {
            saldo -= valor;
            System.out.println("Saque de R$ " + valor + " realizado!");
            System.out.println("Saldo atual: R$ " + saldo);
        } else if (valor <= 0) {
            System.out.println("❌ Valor de saque inválido!");
        } else {
            System.out.println("❌ Saldo insuficiente!");
        }
    }

    public void exibirSaldo() {
        System.out.println("Titular: " + titular);
        System.out.println("Número: " + numero);
        System.out.println("Saldo: R$ " + saldo);
    }

    // ========== MAIN PARA TESTE ==========
    public static void main(String[] args) {
        DesafioConta conta = new DesafioConta();

        System.out.println("--- TESTE: Operações válidas ---");
        conta.setTitular("Maria Santos");
        conta.setNumero(12345);
        conta.exibirSaldo();

        System.out.println("\n--- DEPOSITANDO ---");
        conta.depositar(1000);

        System.out.println("\n--- SACANDO ---");
        conta.sacar(300);

        System.out.println("\n--- TESTE: Operações INVÁLIDAS ---");
        conta.depositar(-50);      // ❌ Bloqueado!
        conta.sacar(1000);         // ❌ Saldo insuficiente!
        conta.exibirSaldo();
    }
}
🎯 O QUE FOI APRENDIDO:
  • ✅ Saldo privado - só pode ser alterado por métodos
  • ✅ Validação de depósitos (valor > 0)
  • ✅ Validação de saques (saldo suficiente)
  • ✅ Setter apenas para dados que podem ser alterados (titular, número)
  • ✅ Getter para ler saldo, sem permitir alteração direta
💡 DICA IMPORTANTE: O atributo saldo NÃO tem setter! Isso garante que NINGUÉM possa definir o saldo diretamente. A única forma de alterar o saldo é através dos métodos depositar e sacar.
