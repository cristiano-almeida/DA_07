📖 O que é Programação Orientada a Objetos?

POO é um paradigma de programação que se baseia na criação de objetos que combinam dados (atributos) e comportamentos (métodos).

🎯 ANALOGIA PARA ENTENDER:
  • 📦 Classe: É o molde (ex: projeto de um carro)
  • 🔷 Objeto: É o carro real construído a partir do molde
  • 📊 Atributos: São as características (cor, modelo, ano)
  • ⚡ Métodos: São as ações (ligar, acelerar, frear)

🎯 Por que usar POO?

  • 📦 Organização: Código mais limpo e estruturado
  • 🔄 Reutilização: Classes podem ser reaproveitadas
  • 🔒 Segurança: Dados protegidos (encapsulamento)
  • 📈 Escalabilidade: Facilita a manutenção e evolução

📊 Exemplo do Mundo Real

🏍️ Motocicleta:
  • ATRIBUTOS (TEM): modelo, marca, cor, placa, chassi
  • MÉTODOS (FAZ): licenciar, checarChassi, ligar
  • ESTADO (COMO ESTÁ): nova, usada, quebrada

🏛️ Os 4 Pilares da POO

📊 1. Abstração

Usar coisas simples para representar a complexidade.

Exemplo: Você sabe ligar a TV, mas não precisa saber como funciona internamente.

Em Java: classes, objetos e variáveis representam códigos complexos.

🔒 2. Encapsulamento

Manter os dados privados e fornecer acesso por métodos públicos.

Exemplo: Os dados de um banco são privados, mas você pode acessá-los com senha.

Em Java: private para atributos e public para métodos.

📦 3. Herança

Criar novas classes que compartilham atributos de classes existentes.

Exemplo: Carro herda de Veiculo (tem rodas, motor, etc.)

Em Java: usa-se extends para herdar.

🔄 4. Polimorfismo

Usar a mesma palavra para significar coisas diferentes em contextos diferentes.

Exemplo: O método falar() pode ser de Cachorro ou Gato.

Em Java: sobrecarga e sobrescrita de métodos.

💡 Neste curso: Vamos focar nos conceitos de Abstração e Encapsulamento.

📦 Classes em Java

Uma classe é o molde para criar objetos. Ela define os atributos e métodos que os objetos terão.

📝 Sintaxe de uma Classe

package tema07;

public class NomeDaClasse {
    // ATRIBUTOS (características)
    tipo nomeAtributo;

    // MÉTODOS (comportamentos)
    public void nomeMetodo() {
        // código do método
    }
}

🏍️ Exemplo: Classe Motocicleta

package tema07;

public class Motocicleta {
    // ATRIBUTOS
    String modelo;
    String marca;
    String cor;
    int cilindradas;

    // MÉTODOS
    public void ligar() {
        System.out.println("Moto ligada!");
    }

    public void desligar() {
        System.out.println("Moto desligada!");
    }
}
📌 OBSERVAÇÃO: A classe é o molde. Ela NÃO ocupa espaço na memória até ser instanciada (criar um objeto).

🔷 Objetos em Java

Um objeto é uma instância de uma classe. Ele ocupa espaço na memória e tem valores reais para os atributos.

📝 Como criar (instanciar) um objeto

// Declarar e instanciar um objeto
NomeDaClasse nomeObjeto = new NomeDaClasse();

// Exemplo com Motocicleta
Motocicleta moto = new Motocicleta();

✏️ Como atribuir valores (atributos)

// Atribuindo valores
moto.modelo = "CG 125";
moto.marca = "Honda";
moto.cor = "Vermelha";
moto.cilindradas = 125;

// Exibindo valores
System.out.println("Modelo: " + moto.modelo);
System.out.println("Marca: " + moto.marca);
System.out.println("Cor: " + moto.cor);

⚡ Como chamar métodos

// Chamando métodos
moto.ligar();    // "Moto ligada!"
moto.desligar(); // "Moto desligada!"

public class ExemploAluno {

📌 Exemplo 2: Classe ContaBancaria (Encapsulamento)

🎯 OBJETIVO: Criar uma classe com encapsulamento (atributos privados e métodos públicos). 
package tema07;

public class ExemploConta {
    // Atributos privados
    private String titular;
    private double saldo;

    // Métodos públicos (getters e setters)
    public String getTitular() {
        return titular;
    }

    public void setTitular(String titular) {
        this.titular = titular;
    }

    public double getSaldo() {
        return saldo;
    }

    public void depositar(double valor) {
        if (valor > 0) {
            saldo += valor;
            System.out.println("Depósito de R$" + valor + " realizado!");
        } else {
            System.out.println("Valor inválido!");
        }
    }

    public void sacar(double valor) {
        if (valor > 0 && valor <= saldo) {
            saldo -= valor;
            System.out.println("Saque de R$" + valor + " realizado!");
        } else {
            System.out.println("Saldo insuficiente ou valor inválido!");
        }
    }

    public void exibirSaldo() {
        System.out.println("Titular: " + titular);
        System.out.println("Saldo: R$" + saldo);
    }

    public static void main(String[] args) {
        ExemploConta conta = new ExemploConta();

        conta.setTitular("Maria Santos");
        conta.depositar(1000);
        conta.sacar(300);
        conta.exibirSaldo();
    }
}
📤 Saída:
Depósito de R$1000.0 realizado!
Saque de R$300.0 realizado!
Titular: Maria Santos
Saldo: R$700.0

📌 Exemplo 3: Classe Carro (Main + Classe em arquivos separados)

🎯 OBJETIVO: Demonstrar o uso de classes separadas em arquivos diferentes:
📄 Carro.java → define a classe (molde)
📄 TesteCarro.java → contém o main e usa a classe Carro
📌 ESTRUTURA: Crie dois arquivos no pacote tema07:
  • 📄 Carro.java → contém a definição da classe
  • 📄 TesteCarro.java → contém o método main

📄 Carro.java (Classe)

package tema07;

public class Carro {
    // Atributos
    String modelo;
    String cor;
    int velocidade;

    // Método para acelerar
    public void acelerar() {
        velocidade += 10;
        System.out.println("Acelerando... Velocidade: " + velocidade + " km/h");
    }

    // Método para frear
    public void frear() {
        if (velocidade >= 5) {
            velocidade -= 5;
            System.out.println("Freando... Velocidade: " + velocidade + " km/h");
        } else {
            System.out.println("Carro já está parado!");
        }
    }

    // Método para exibir status
    public void exibirStatus() {
        System.out.println("Modelo: " + modelo);
        System.out.println("Cor: " + cor);
        System.out.println("Velocidade atual: " + velocidade + " km/h");
    }
}

📄 TesteCarro.java (Main)

package tema07;

public class TesteCarro {
    public static void main(String[] args) {
        // Criando um objeto da classe Carro
        Carro meuCarro = new Carro();

        // Atribuindo valores
        meuCarro.modelo = "Fusca";
        meuCarro.cor = "Azul";
        meuCarro.velocidade = 0;

        // Exibindo status inicial
        System.out.println("=== STATUS INICIAL ===");
        meuCarro.exibirStatus();

        // Testando métodos
        System.out.println("\n=== TESTANDO ACELERAÇÃO ===");
        meuCarro.acelerar();
        meuCarro.acelerar();
        meuCarro.acelerar();

        System.out.println("\n=== TESTANDO FRENAGEM ===");
        meuCarro.frear();
        meuCarro.frear();

        System.out.println("\n=== STATUS FINAL ===");
        meuCarro.exibirStatus();
    }
}
📤 Saída:

Modelo: Fusca
Cor: Azul
Velocidade atual: 0 km/h

=== TESTANDO ACELERAÇÃO ===
Acelerando... Velocidade: 10 km/h
Acelerando... Velocidade: 20 km/h
Acelerando... Velocidade: 30 km/h

=== TESTANDO FRENAGEM ===
Freando... Velocidade: 25 km/h
Freando... Velocidade: 20 km/h

=== STATUS FINAL ===
Modelo: Fusca
Cor: Azul
Velocidade atual: 20 km/h
