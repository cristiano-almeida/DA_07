🧩 Desafio 1: Classe Livro

📌 Classe: DesafioLivro.java
📌 Atributos: titulo, autor, anoPublicacao
📌 Método: exibirInfo() para exibir os dados
📌 main na própria classe para criar e testar um objeto
📚 TRÍADE DIDÁTICA:
  • CONCEITO: Classe é o molde, objeto é a instância.
  • OBJETIVO: Criar um livro com título, autor e ano, e exibir suas informações.
  • MÉTODO: Atributos públicos (default) e método exibirInfo().
📤 Saída esperada:
--- DADOS DO LIVRO ---
Título: Java para Iniciantes
Autor: Maria Silva
Ano de Publicação: 2023

📝 Código Sugerido:

package tema07;

public class DesafioLivro {
    // ATRIBUTOS
    String titulo;
    String autor;
    int anoPublicacao;

    // MÉTODO PARA EXIBIR INFORMAÇÕES
    public void exibirInfo() {
        System.out.println("Título: " + titulo);
        System.out.println("Autor: " + autor);
        System.out.println("Ano de Publicação: " + anoPublicacao);
    }

    // MÉTODO MAIN PARA TESTE
    public static void main(String[] args) {
        // Criando um objeto da classe Livro
        DesafioLivro livro = new DesafioLivro();

        // Atribuindo valores
        livro.titulo = "Java para Iniciantes";
        livro.autor = "Maria Silva";
        livro.anoPublicacao = 2023;

        // Exibindo informações
        System.out.println("--- DADOS DO LIVRO ---");
        livro.exibirInfo();
    }
}

  • ✅ Chamar métodos do objeto

🧩 Desafio 2: Classe Produto (com encapsulamento)

📌 Classe: DesafioProduto.java
📌 Atributos privados: nome, preco
📌 Getters e Setters para cada atributo
📌 Método aplicarDesconto(double percentual) com validação (0-100%)
📌 main com testes de valores válidos e inválidos
📚 TRÍADE DIDÁTICA:
  • CONCEITO: Encapsulamento protege os dados do produto.
  • OBJETIVO: Garantir que nome e preço sejam sempre válidos.
  • MÉTODO: Atributos private e setters com validação.
📤 Saída esperada:
--- TESTE: Valores válidos ---
Produto: Notebook
Preço: R$ 2500.0

--- APLICANDO DESCONTO ---
Desconto de 15% aplicado!
Novo preço: R$ 2125.0

--- TESTE: Valores INVÁLIDOS ---
❌ Nome inválido!
❌ Preço inválido! Deve ser maior que zero.
❌ Desconto inválido! Use entre 0 e 100.

📝 Código Sugerido:

package tema07;

public class DesafioProduto {
    // ATRIBUTOS PRIVADOS
    private String nome;
    private double preco;

    // ========== GETTERS ==========
    public String getNome() {
        return nome;
    }

    public double getPreco() {
        return preco;
    }

    // ========== SETTERS (com validação) ==========
    public void setNome(String nome) {
        if (nome != null && !nome.trim().isEmpty()) {
            this.nome = nome;
        } else {
            System.out.println("❌ Nome inválido!");
        }
    }

    public void setPreco(double preco) {
        if (preco > 0) {
            this.preco = preco;
        } else {
            System.out.println("❌ Preço inválido! Deve ser maior que zero.");
        }
    }

    // ========== MÉTODOS ==========
    public void aplicarDesconto(double percentual) {
        if (percentual >= 0 && percentual <= 100) {
            double desconto = preco * (percentual / 100);
            preco -= desconto;
            System.out.println("Desconto de " + percentual + "% aplicado!");
            System.out.println("Novo preço: R$ " + preco);
        } else {
            System.out.println("❌ Desconto inválido! Use entre 0 e 100.");
        }
    }

    public void exibirInfo() {
        System.out.println("Produto: " + nome);
        System.out.println("Preço: R$ " + preco);
    }

    // ========== MAIN PARA TESTE ==========
    public static void main(String[] args) {
        DesafioProduto produto = new DesafioProduto();

        System.out.println("--- TESTE: Valores válidos ---");
        produto.setNome("Notebook");
        produto.setPreco(2500.0);
        produto.exibirInfo();

        System.out.println("\n--- APLICANDO DESCONTO ---");
        produto.aplicarDesconto(15);

        System.out.println("\n--- TESTE: Valores INVÁLIDOS ---");
        produto.setNome("");          // ❌ Bloqueado!
        produto.setPreco(-100);       // ❌ Bloqueado!
        produto.aplicarDesconto(150); // ❌ Bloqueado!
    }
}
🎯 O QUE FOI APRENDIDO:
  • ✅ Uso de private para proteger atributos
  • ✅ Getters para ler valores com segurança
  • ✅ Setters com validação para evitar dados inválidos
  • ✅ Método com validação (aplicarDesconto)

  • OBJETIVO: Demonstrar como organizar o código com classes separadas.
  • MÉTODO: Classe Carro em um arquivo, TesteCarro em outro (com main).

📄 DesafioCarro.java (Classe)

package tema07;

public class DesafioCarro {
    // ATRIBUTOS
    String modelo;
    String cor;
    int velocidade;

    // MÉTODO PARA ACELERAR
    public void acelerar() {
        velocidade += 10;
        System.out.println("Acelerando... Velocidade: " + velocidade + " km/h");
    }

    // MÉTODO PARA FREAR
    public void frear() {
        if (velocidade >= 5) {
            velocidade -= 5;
            System.out.println("Freando... Velocidade: " + velocidade + " km/h");
        } else {
            System.out.println("Carro já está parado!");
        }
    }

    // MÉTODO PARA EXIBIR STATUS
    public void exibirStatus() {
        System.out.println("Modelo: " + modelo);
        System.out.println("Cor: " + cor);
        System.out.println("Velocidade atual: " + velocidade + " km/h");
    }
}

📄 TesteDesafioCarro.java (Main)

package tema07;

public class TesteDesafioCarro {
    public static void main(String[] args) {
        // Criando um objeto da classe Carro
        DesafioCarro meuCarro = new DesafioCarro();

        // Atribuindo valores
        meuCarro.modelo = "Fusca";
        meuCarro.cor = "Azul";
        meuCarro.velocidade = 0;

        // Exibindo status inicial
        System.out.println("=== STATUS INICIAL ===");
        meuCarro.exibirStatus();

        // Testando aceleração
        System.out.println("\n=== TESTANDO ACELERAÇÃO ===");
        meuCarro.acelerar();
        meuCarro.acelerar();
        meuCarro.acelerar();

        // Testando frenagem
        System.out.println("\n=== TESTANDO FRENAGEM ===");
        meuCarro.frear();
        meuCarro.frear();

        // Exibindo status final
        System.out.println("\n=== STATUS FINAL ===");
        meuCarro.exibirStatus();
    }
}
🎯 O QUE FOI APRENDIDO:
  • ✅ Organização do código com classes separadas
  • ✅ Classe sem main (apenas definição)
  • ✅ Classe com main separado para testar a classe
  • ✅ Como usar uma classe em outro arquivo (mesmo pacote)
  • ✅ Métodos que modificam o estado do objeto (acelerar, frear)
